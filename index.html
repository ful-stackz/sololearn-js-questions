This app is still under development! It is not fully functional and the question base is small - for now you can face only 13 different questions.

Nevertheless, if there's interest I will keep developing this app to add more functionality. For now the only type of questions supported are single choice answerable; I'm planning to add multiple-answers questions and user-typed answers as soon as possible. # About

This is the place to go Solo on SoloLearn's JavaScript challenge questions!

This app allows you to go Single Player against some of the JavaScript challenge questions from SoloLearn. First, go to the exercise section (the lightning icon on the top) and press start. Then the app will ask you a serie of 5 questions, just like in a SoloLearn challenge. After the 5 answers, you will see the correct answers of the questions you answered wrong and an explanation (if there is). The app will keep a record of your correctly and incorrectly answered questions. You can then visit the profile section (the little head icon) to review your stats. However, due to the lack of database the statistics will be lost, once you close the app.

# Contribution

SoloLearn is a huge platform! This tiny trainer needs your help!

In order to expand this exercising app I need your help! If you would like to contribute with anything you're more than welcome! For now I'm looking forward to expanding the database of questions. I will include as much as I can get from SoloLearn, because afterall the purpose is to practice for the challenges, right? But, you are welcome to submit your own questions in the comments. The question object this app is using is a very simple JSON object.
{
   question: 'The text of the question goes here',
   wrong: [ 'This is an array with all the', 'wrong', 'answers'],
   correct: 'Here comes the correct answer',
   time: 10, // time is a Number, indicates the time you'll have to answer the question, in seconds
   explanation: 'This is some text. Here may come an explanation to why the answer is what it is.'
}
